## Stark-UI

Stark UI is a CSS component library. Made to use beautiful UI components in your web projects to make development process easier and more efficient.

# Technologies Used

    -HTML
    -CSS

# list of Components

-Avatar
-Inputs
-Buttons
-Badges
-List
-Cards
-Images
-Nav bar
-Typography
-Toast
